## Разворачивание проекта
В консоли поочередно выполнить следующие команды:
<ul>
    <li>git clone</li>
    <li>cd cdr</li>
    <li>composer i</li>
    <li>npm i</li>
    <li>php artisan jwt:secret</li>
</ul>

Далее необходимо настроить .env:
<ul>
    <li>Подключение к БД системы</li>
    <li>TOKEN_REFRESH</li>
</ul>
